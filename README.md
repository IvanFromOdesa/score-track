# Score-track
## Introducing Score-track: Your Ultimate Free Sports Stats-Tracker!
### Tech stack:
- Java 17
- Spring Boot v.3.1.2
- Hibernate v.6.2.6
- PostgreSQL
- MongoDB
- Redis
- Spring Security v.3.1.2
- Thymeleaf v.3.1.2
- React
- Google cloud storage
- Docker

Additional libraries used in project can be found at ```pom.xml```.
